## Uso

Para correr los ejemplos, se debe utilizar un compilador de C.
En Linux, se puede utilizar GCC.

```bash
sudo apt update
sudo apt install build-essential
gcc <nombre_del_archivo_fuente>.c -o <nombre_del_ejecutable>
```
